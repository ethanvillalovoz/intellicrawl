# Code of Conduct

## Our Pledge

Contributors and maintainers agree to keep this project respectful, constructive, and focused on useful technical discussion.

## Expected Behavior

- Be professional and helpful in issues, pull requests, and reviews.
- Keep criticism focused on code, documentation, architecture, and reproducibility.
- Respect that API behavior, pricing, and model output can change over time.
- Avoid sharing private keys, private scraped content, or confidential user data.

## Unacceptable Behavior

- Harassment, insults, personal attacks, or disruptive comments.
- Publishing another person's private data, credentials, or confidential content.
- Repeated off-topic arguments that prevent maintainers from resolving issues.

## Enforcement

Maintainers may edit, hide, or remove comments and close issues or pull requests that violate this code of conduct.
